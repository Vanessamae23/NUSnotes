Java Basics

Java Static

Static Variable

  • Static variables can be accessed by calling with the class name ClassName.VariableName.
  • Static variables are declared with the static keyword in a class, but outside a method, constructor or a block.
  • There would only be one copy of each class variable per class, regardless of how many objects are created from it.

Static Class

  • The static keyword is used to create variables that will exist independently of any instances created for the class. Only one copy of the static variable exists regardless of the number of instances of the class.

Static Method

  • Static method DO NOT use any instance variables of ANY OBJECT of the class they are defined in but they can take data from static fields and all data from the parameters
public class MyClass {
    public static void myStaticMethod(MyClass obj) {
        int y = this.instanceVariable; // cannot be referenced from a static context
        int x = obj.instanceVariable; // valid because it is the parameter instance variable
        obj.instanceMethod();
    }

    public int instanceVariable = 42;

    public void instanceMethod() {
        System.out.println("Hello from instanceMethod!");
    }
}

Java Final

Final Variables

  • A final variable can be explicitly initialized only once. A reference variable declared final can never be reassigned to refer to a different object
  • Data within the object can be changed so the state can be changed but not the reference
  • Example: public static final Maybe<?> NONE = new None();

Final Methods

  • A final method cannot be overridden by any subclasses. As mentioned previously, the final modifier prevents a method from being modified in a subclass.
  • The main intention of making a method final would be that the content of the method should not be changed by any outsider.

Final Class

  • The main purpose of using a class being declared as final is to prevent the class from being sub-classed / inherited. If a class is marked as final then no class can inherit any feature from the final class.
